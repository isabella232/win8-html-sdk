This sample shows how you can use RadAutoCompleteBox to filter the data displayed in another control, in this case, a ListView. To see it working start typing in a company name from Northwind's Customers data, e.g. "Bon App'":

This is achieved by hooking up the databound event of RadAutoCompleteBox and assigning its own data as an item data source for the ListView. Additonally, the open event of the autocomplete is handled to prevent its dropdown from opening.
