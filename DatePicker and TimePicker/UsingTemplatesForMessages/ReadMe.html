This sample shows how you can use string templates for the messages inside the picker controls. This is useful when the value you want for the picker comes from another place and is wrapped in an object.

To use a value from an object to populate the picker strings, assign the object to the corresponding part of the control: header, emptyContent, etc. and set the template property (headerTemplate, emptyContentTemplate, etc.) using binding expressions (#=object.fieldName#).
