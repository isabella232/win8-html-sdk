This sample shows a hierarchical RadGrid, using declarative relations between the tables. Click on an expand button to view the detail table for each row.

To create a hierarchical RadGrid, you need to define a detailTable object in the RadGrid constructor and assign it a data source and parent relation. The relation should list two fields: masterField which is the field from the master grid used for the relation and detailField which is the field from the detail grid that links to the master field.
