This example demonstrates how a databound RadControl can be added inside a declarative WinJs.Binding.Template. The template depends on a custom binding initializer to assign the properties of the RadChart bound inside the template body. Custom binding initializers allow you to implement more complex binding scenarios where assigning a single value to a property is not enough to achieve your goal.
